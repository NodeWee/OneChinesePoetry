# One Chinese Poetry
一首诗词，一个 JSON 文件，一个独立 URL。可直接通过 HTTP 读取单首诗词内容。方便各种终端的调用。

- 诗词来源：[chinese-poetry 项目](https://github.com/chinese-poetry/chinese-poetry)
